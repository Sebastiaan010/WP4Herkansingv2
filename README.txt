Hoe start je het programma?

Als je bestanden hebt ingeladen in je IDE, open je de terminal en typ je : npm start

Als je dit hebt gedaan open je een nieuwe terminal, in deze terminal typ je : cd client
hierdoor kom je in de folder terecht van de client
als je in deze folder zit typ je opnieuw npm start

de terminal zal dan vragen of je de applicatie wilt runnen op een andere port, dit omdat je server al op port 3000 loopt
je klikt dan op Y en dan opent die de applicatie op 3001.

je gaat nu naar http://localhost:3001/ in je browser en nu kan je de applicatie gebruiken
